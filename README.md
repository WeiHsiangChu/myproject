# Wei-Hsiang Chu's Project Portfolio

This repository contains selected works from my undergraduate and graduate studies in Electrical and Computer Engineering, ranging from antenna design to Machine Learning, Deep Learning, and Reinforcement Learning projects.

## 📘 Undergraduate Projects  

- **[Laptop Antenna Design for Wi-Fi 6E Band](undergraduate/antenna-design)**  
  Designed a PIFA antenna for use in narrow-bezel laptops, optimized for the Wi-Fi 6E band.

## 🎓 Graduate Projects  

- **[Gesture Recognition using WeBe Band](graduate/Gesture%20Detection)**
  Built an embedded machine learning pipeline for real-time gesture recognition using wrist-worn accelerometer data, achieving 98.1% classification accuracy.

- **[Fall Detection using WeBe Band](graduate/Fall%20Detection)**
  Developed a wrist-based fall detection system using signal processing, threshold optimization, and LOOCV evaluation for wearable healthcare applications.

- **[AI for Early Detection of Arrhythmia](graduate/arrhythmia-detection)**
  Applied supervised learning techniques to detect early signs of arrhythmia from physiological signals.

- **[ATE Drift Prediction](graduate/ATE-drift-prediction)**
  Simulated an ATE environment and trained agents to predict equipment drift using Double Q-Learning.

### 📂 Course Projects (RL & ML)  

- **[MNIST Patch K-Means](graduate/course-projects/mnist-patch-kmeans)**  
  K-means on 5×5 MNIST patches; cluster evolution for different K and patch reconstruction.  

- **[Non-Parametric Data Synthesis](graduate/course-projects/non-parametric-data-synthesis)**  
  Efros & Leung (1999) texture synthesis on 32×32 and 64×64 textures, extended to MNIST digit generation.  

- **[Few-Shot Representation Learning](graduate/course-projects/few-shot-representation-learning)**  
  Self-supervised pretraining on Oxford 102 Flowers. Compared SimCLR vs MAE with linear/MLP probes.  

- **[Gesture Detection (Contrastive Learning)](graduate/course-projects/gesture-detection-contrastive-learning)**  
  SimCLR on a two-class gesture subset (**circle vs eight**) using preprocessed sequence features; compared X_feat, X_crop, X_resampling (resampling performed best).  

- **[MCES Blackjack](graduate/course-projects/mces-blackjack)**  
  Monte Carlo Exploring Starts for Blackjack; reproduced policy (π*) and value (V*) under dealer soft-17 variations with/without usable ace.  

- **[Cliffwalking TD](graduate/course-projects/cliffwalking-td)**  
  TD control in the 4×12 Cliffwalking domain; compared Sarsa, Q-learning, and Expected Sarsa.  

- **[Random Walk](graduate/course-projects/random-walk)**  
  1000-state random walk; compared Gradient Monte Carlo vs Semi-Gradient TD(0) with state aggregation.  

- **[Multi-Armed Bandit](graduate/course-projects/multiarmed-bandit)**  
  Epsilon-greedy vs UCB on k-armed bandits; reproduced standard textbook comparison figure.  

- **[Coin Toss – Optimal Policy](graduate/course-projects/coin-toss-optimal-policy)**  
  Optimal stopping policy analysis for the coin-toss setting.  

- **[Coin Toss – Policy Evaluation](graduate/course-projects/coin-toss-policy-evaluation)**  
  Empirical evaluation of candidate coin-toss stopping policies.
